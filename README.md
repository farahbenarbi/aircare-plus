# AirCare+

Systeme IoT de surveillance de la qualite de l'air : capteurs environnementaux, 
prediction de risques sanitaires par IA, et engagement gamifie autour de la durabilite.

## Materiel
- ESP32 DevKit V1
- Capteur DHT22 (temperature/humidite)
- Breadboard, jumpers, resistance 220 ohms, LED

## Installation
1. Installer Arduino IDE ou PlatformIO
2. Ouvrir sketch.ino
3. Selectionner la carte ESP32 Dev Module
4. Televerser sur la carte (ou simuler sur Wokwi)
